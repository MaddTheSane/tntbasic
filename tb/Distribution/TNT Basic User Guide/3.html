Chapter 3

Hieroglyph

Introducing Hieroglyph

Hieroglyph is used to edit TNT Basic project files. It provides basic resource manipulation menu commands that are common to most of the sections in the project. These are:

Resource Menu : Create new resource...
This will create a new resource in the current section. In the sounds, music and graphics section you will be prompted to select a sound/music/graphics file to import. In the maps and input section a new resource will be created an an editor opened on it so you can edit it.
Creating resources in each of the section is discussed in greater depth in the sections' guide later in this document.

Resource Menu : Get Info...
This command allows you to edit the name and id of the selected resource(s).

Resource Menu : Save/Revert
These two commands are enabled when you are editing a resource. Save saves the changes you have made to the resource into the project file. Revert reverts back to the last saved copy of the resource.

Copying Resources

Resources can be copied between projects using Hieroglyph.
Simply open two files at the same time and drag the resources from one file to another to copy them. You can also use copy and paste from Hieroglyph "Edit" menu if you prefer.

A resource can be duplicated by selecting it and choosing "Duplicate" from the resource menu. The resource will automatically be assigned a new and unqiue resource ID.

Section Guide

It was previously explained how the TNT Basic project file was split into seven sections as follows:

  • Info Section
  • Code Section
  • Graphics Section
  • Sounds Section
  • Music Section
  • Maps Section
  • Input Section

The rest of this chapter will explain the significance of each section and how it is edited in Hieroglyph. At the end of the chapter is an explanation of the commands in the TNT Basic menu.
