What is TNT Basic?

TNT Basic is a games development kit for the Macintosh that allows anyone, regardless of previous experience, to make their own great games easily!

How?

Making games in TNT Basic involves programming in a simple language called basic, which is designed especially for beginners. Don't worry if you can't program, we'll show you how, anyone can do it with TNT Basic!

Make your own games today!

Technology

Even though we've made it easier than ever to create games, TNT Basic doesn't hide from you the technology you'll need to make great games. TNT Basic uses simple terms and commands to give you access to the following great technologies:

  • 16 bit graphics (Over 32,000 colours on screen!)
  • Lightning fast sprite engine (supports anti-aliasing and translucency!)
  • Stereo panning sound support
  • Advanced music engine (supports MP3 and many tracker music formats including MOD, MAD, MIDI, SoundTracker and OctaMed)
  • TCP/IP and AppleTalk game networking - play your friends over the internet!
  • Build standalone games which don't need TNT Basic to run
  • Load and save hi-scores and saved games
  • Automatic joystick, mouse, keyboard and speech input support
  • Integrated development environment, manage everything from code to music from one application!
  • Syntax colouring, your code is coloured automatically to make it easier to read
  • Integrated help, just highlight a command in your code and hit cmd-? to find out more about it
  • Mac OS X compatible

What's in the Package?

In the TNT Basic package you will find the following files:

  • Hieroglyph
  • Hieroglyph Blocks
  • TNT Basic
  • TNT Basic Help
  • TNT Basic User Guide
  • Examples
  • Report Bug (OS X) & Report Bug (OS 9)
  • Various Text Files

Hieroglyph
Hieroglyph is TNT Basic's editor. You use Hieroglyph to edit your TNT Basic projects and compose all your graphics, sounds, maps, music and code. The resultant project is ran by TNT Basic.

Hieroglyph Blocks
This folder contains plug-ins for Hieroglyph to allow it to work with TNT Basic.

TNT Basic
This is TNT Basic itself. TNT Basic runs your projects and plays your games. You can run TNT Basic yourself and use the "Open..." command to run projects manually if you wish.

TNT Basic Help
The help system for TNT Basic, you can access this from the help menu in TNT Basic and Hierogyph.

TNT Basic User Guide
This user guide. You can access this from the help menu in TNT Basic and Hieroglyph.

Examples
Example games and demos for TNT Basic.

Report Bug (OS X) and Report Bug (OS 9)
If you find any problems with TNT Basic or Hieroglyph, which you think is our fault, please use these bug reporting scripts to report them to us. They will gather information from TNT Basic and Apple's System Profiler, which you should submit with your bug report to TNT Basic's bug tracker.
