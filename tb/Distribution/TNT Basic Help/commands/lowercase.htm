Lower Case

Converts a string into entirely lower case letters.

Syntax

string = Lower Case (string)

Description

This command converts the string passed into a string consisting of just lower case letters.

e.g. The value "Hello" would become the string "hello".

This command is useful for comparing strings without case.

if "Hello"="hello"
...
end if

The body of this if construct is not executed because the two 'hello' strings are not equal.

if Lower Case("Hello")=Lower Case("hello")
...
end if

The body of this if construct is executed because the case is removed from both strings.
