As String

Converts a number into a string.

Syntax

string = As String (number, numdp)

Description

This converts the number passed into a string.

number is the number to be converted to a string.

numdp is the number of decimal places to use when converting the string. This is particularly useful when passing floating point values to the function.

e.g. The value 12 would become the string "12".
The value 4.5 would become the string "4.5".
