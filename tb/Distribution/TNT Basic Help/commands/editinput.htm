Edit Input

Displays the dialog for setting input preferences.

Syntax

Edit Input

Description

At the present time, this command does not work on Mac OS X, due to its reliance on Input Sprocket which has been discontinued.

This command displays a dialog which allows the user to edit the controls for the game. All the users preferences are saved under the current creator code. As long as the creator code for your game is not changed then the preferences will be saved.
