File Read

Reads data from a file

Syntax

result = File Read Int (fileId)
result = File Read Float (fileId)
result = File Read String (fileId)

Description

This command reads data from the specified file.

fileId is the id of the file obtained from Ask Open File or Open File. If this is invalid then this command will cause an error.

result can be an integer, float or string depending on which command was called.
