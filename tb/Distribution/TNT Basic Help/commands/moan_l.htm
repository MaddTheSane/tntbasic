'L'et

Assign register with a number

Syntax

L Rx = exp

Description

Sets register Rx to exp.

Examples:

"L R0 = 7" ' Let local register 0 equal 7
"L RA = R0" ' Let shared register A equal local register 0
"L R3 = 7+(2^5)" ' Let local register 3 equal 39

See also the Set Channel Register command.
