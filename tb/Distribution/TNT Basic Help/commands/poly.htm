Poly

Draw a polygon at the specified co-ordinates.

Syntax

Fill Poly x1,y1 to x2,y2 to ......
Frame Poly x1,y1 to x2,y2 to.....

Description

A polygon can be any shape or size, it is just any shape that is specifed by the series of points in the co-ord list.

Fill Poly draws a solid polygon between all the co-ordinates specified.

Frame Poly draws a hollow polygon (outline) between all the co-ordinates specified.

Up to 50 co-ordinates can be specified in the co-ordinate list.
