Host Net Game

Displays a dialog allowing the user to gather players into a game.

Syntax

bool = Host Net Game (maxPlayers,gName,pName)

Description

This command allows the user to create a network game.

maxPlayers is the maximum number of players that can join the game.

gName is the default name for the game when it is advertised on the network. This can be changed by the user in the host dialog.

pName is the default name for the player in the game. This can be changed by the user in the host dialog. If an empty string is passed here then TNT Basic will use the machine owners name from the system.

The bool returned is a boolean containing whether or not the user decided to start the game.

False = Game Cancelled or User Kicked Out
True = Game Start
