Set Sprite X Scale

Sets the horizontal scale of a sprite.

Syntax

Set Sprite X Scale spriteID,scale

Description

The x scale of a sprite controls the width of the sprite. This is applied before the rotation.

spriteID is the number of the sprite.

scale is the scale adjustment the sprite. This value is a floating point mutliplier so if you pass 2 it will double the width of the sprite, if you pass 0.5 it will half the width of the sprite.

Note: This function only works in hardware mode.
