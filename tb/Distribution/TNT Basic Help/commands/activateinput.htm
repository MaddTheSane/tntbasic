Activate Input

Switches to input mode.

Syntax

Activate Input

Description

At the present time, this command does not work on Mac OS X, due to its reliance on Input Sprocket which has been discontinued.

There a two modes that the input can be read in from. In the normal mode the mouse and keys can be read from. In input mode the Poll Input command can be used but everything else will not work. This command switches to input mode so that the Poll Input command can be used.

This can be reversed with a call to Deactivate Input.
