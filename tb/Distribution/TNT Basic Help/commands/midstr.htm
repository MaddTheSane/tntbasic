Mid Str

Gets characters from the middle of a string

Syntax

string = Mid Str (theString,start,length)

Description

Returns length characters from the middle of theString starting at start characters into the string.
The first character in the string is at index 1.

Example:
str sub
str test = "This is my test string"
sub = mid str (test,2,3)
print sub

This would output 'his'.

See also Left Str, Right Str and Get Char.
