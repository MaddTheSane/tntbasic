Map Line Col

Tests if a specified line contains certain tiles.

Syntax

bool=Map Line Col(x1,y1 to x2,y2,layer,start to end)

Description

This command tests if any tile between start and end inclusive lies on the line specified by x1,y1 to x2,y2. It only tests in the layer specified by layer.

bool is true if any of the tiles did lie on the line, otherwise it is false.
