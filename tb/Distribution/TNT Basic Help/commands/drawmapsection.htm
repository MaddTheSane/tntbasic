Draw Map Section

Draw a specified section of the map

Syntax

Draw Map Section x1,y1 to x2,y2,layer to x,y

Description

This command draws the specified area of the map to the current canvas.

x1,y1 is the top-left of the area to draw (in tile co-ordinates). x2,y2 is the bottom-right of the area to draw (in tile co-ordinates).

layer is the layer of the map to draw.

x,y is the top left point to the canvas to draw the map section to.
