Wait Mouse Click

Pauses until the mouse button is clicked.

Syntax

Wait Mouse Click

Description

This command pauses execution of the program until the mouse button is clicked. It is equivalent to writing the following piece of code...

while not Button
wend

while Button
wend

Note: The input has to be in standard mode for this function to work. See Activate Input and Deactivate Input for more information about input modes.
