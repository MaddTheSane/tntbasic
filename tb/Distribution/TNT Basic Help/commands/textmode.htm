Text Mode

Exits graphics mode and enters text mode

Syntax

Text Mode

Description

If executed from inside graphics mode (see Graphics Mode) this command returns TNT Basic to text mode. In text mode a simple input output interface is presented to the user.

Exiting graphics mode frees up all memory used by graphic entities such as sprites, images and canvases.

TNT Basic defaults to being in text mode. If this command is executed from inside text mode it has no effect.
