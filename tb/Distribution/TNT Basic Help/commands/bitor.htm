Bit Or

This command ors the bits of two integers together.

Syntax

result = Bit Or(valueA,valueB)

Description

In TNT Basic, all integers are actually just composed of 32 individual 'bits', a bit can be either a '0' or a '1'.

This command takes two numbers, valueA and valueB, then ors them together to produce the result.

It is possible to set an individual bit using the function Bit Set.
