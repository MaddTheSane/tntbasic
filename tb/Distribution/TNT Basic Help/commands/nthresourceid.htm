Nth Resource ID

Returns the resource id of the nth resource of that type.

Syntax

id = Nth Resource ID (type,index)

Description

This command is mainly used when loading resources.

In order to load a resource you need its id. To obtain the id of a resource this command can be used. e.g.

Load Map Nth Resource ID ("TMAP",4)

This will load the 5th Map in the resource file (since the index is from 0).
