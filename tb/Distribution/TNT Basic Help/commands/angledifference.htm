Angle Difference

Calculates the difference between two angles.

Syntax

value = Angle Difference(angleA,angleB)

Description

This command takes two angles and calculates the difference between them. The angle difference is always between -180 and 180.

e.g.

Angle Difference(0,90)=-90
Angle Difference(90,0)=90
Angle Difference(0,360)=0
Angle Difference(0,180)=-180
Angle Difference(0,181)=179
