Restart Channel

Restarts the program on a moan channel

Syntax

Restart Channel
Restart Channel channelId

Description

Restarts a moan channel from the beginning, clearing it's local registers.
Note that all programs affect the sprites from their current location, so often when restarting a program you would want to restore sprite and viewports to their original position (restart channel does not do this).

If channelId is not specified then all channels are restarted.

See also Reset Channel and Set Channel Program.
