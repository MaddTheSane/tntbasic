Set Sprite Rotation

Sets the rotation of a sprite.

Syntax

Set Sprite Rotation spriteID,angle

Description

The rotation of a sprite controls the rotation of it on screen.

spriteID is the number of the sprite.

angle is the clockwise rotation (in degrees) of the sprite. This can be any value including negative numbers (negative numbers are anti-clockwise). The default rotation of a sprite is 0.

Note: This function only works in hardware mode
