Direction Controls

Tests the condition of a key on the keyboard

Syntax

bool = Up
bool = Down
bool = Left
bool = Right
bool = Space

Description

Each of these functions tests a certain key. The directions test the appropriate arrow key and Space tests the space bar.

bool is one of 2 values...

False - The key is not pressed
True - The key is pressed

Note: The input has to be in standard mode for these functions to work. See Deactivate Input and Activate Input for more information about input modes.
