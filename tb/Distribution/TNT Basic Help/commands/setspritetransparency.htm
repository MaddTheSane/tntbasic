Set Sprite Transparency

Sets the transparency level of a sprite.

Syntax

Set Sprite Transparency spriteID,transparency

Description

The transparency setting of a sprite controls how visible it appears. Each sprite can be invisible, opaque or anywhere in between.

spriteID is the number of the sprite.

transparency is the new level of transparency. This can range between 0 (opaque) and 100 (invisible).

There are certain levels of transparency which can be drawn faster than all the others. If speed is critical then we recommend using these transparency levels:
