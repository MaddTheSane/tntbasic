Set Component

Sets the component of a colour to a certain amount.

Syntax

newColour = Set Red Component (colour,component)
newColour = Set Green Component (colour,component)
newColour = Set Blue Component (colour,component)

Description

All colours are composed of red, green and blue components. These values can be between 0 and 255.

Any colour can be built from these commands. e.g.

colour = Set Red Component (colour,255)
colour = Set Green Component (colour,255)
colour = Set Blue Component (colour,255)

This would build white because all components are set to 255.
