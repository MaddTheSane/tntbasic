Image Collision Type

Returns the collision type of a specifed image

Syntax

bool = Image Collision Type (image)
bool = Image Collision Type (image,bankID)

Description

image is the index into the bank the obtain the collision type from.

bankID is the id of the bank to get the image from. If this is not specified then the current bank is used.

bool is one of two values...

True - The image uses mask collision
False - The image only uses rect based collision
