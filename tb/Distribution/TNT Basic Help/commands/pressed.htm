Pressed

Tests the condition of a key on the keyboard

Syntax

bool = Pressed (key)

Description

key is the specific value of the key to test. The codes for each of these are lists below.

bool is one of 2 values...

False - The key is not pressed
True - The key is pressed

Note: The input has to be in standard mode for this function to work. See Deactivate Input and Activate Input for more information about input modes.
