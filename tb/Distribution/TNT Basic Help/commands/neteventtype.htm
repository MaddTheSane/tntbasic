Net Event Type

Gets the type of the current net event.

Syntax

value = Net Event Type

Description

The value returned can be one of the following:

NetGameTerminated
The net game has been terminated by the host.

NetPlayerLeft
A player has left the game. The event content contains the id of the player.

NetPlayerDisconnected
A player has become disconnected. This could be due to network failure or a computer crash. The event content contains the id of the player.
