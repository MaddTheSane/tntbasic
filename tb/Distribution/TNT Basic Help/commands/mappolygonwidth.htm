Map Polygon Width

Returns the total width (in tiles) of the specified polygon in the current map.

Syntax

width = Map Polygon Width (polygonIndex)

Description

width is the total width (in tiles) of the polygon polygonIndex.

Polygons are not necessarily rectangles. This function simply returns the total width from the left side to the right side of the polygon. To test if individual tiles are contained inside the polygon use the function In Map Polygon.
