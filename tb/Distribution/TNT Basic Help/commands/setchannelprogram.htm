Set Channel Program

Installs a moan program into a moan channel

Syntax

Set Channel Program channelId,progString

Description

channelId is the moan channel to install the program on, it can be any number.
progString is a moan program. You may pass an empty string to clear the program.

The programs are written according to the Moan language spec, described in the Moan Tutorial.

Setting the program clears the local registers for the channel.

See also Channel Running, Restart Channel and Reset Channel.
