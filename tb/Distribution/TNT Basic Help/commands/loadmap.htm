Load Map

Load the specified map into memory.

Syntax

Load Map resID

Description

resID can be either an integer or a string. It is the id or the name of the 'TMAP' resource to be loaded.

If the map can not be found then this function will return an error.

Maps can be unloaded from memory (in order not to use too much at one time) using the command Unload Map.

This command automatically sets this map to the currently active map. For more information on current maps see the command Set Current Map.
