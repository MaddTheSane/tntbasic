Set Net Data

This command updates the value of the specified network variable.

Syntax

Set Net Data data,value

Description

This command sends a message to the host telling it to update the specified network variable to the new value. This command is used so that clients can send messages to the host.

data is the variable that you wish to update. It can be an integer, float or a string.

value is the new value of the variable. This can be an interger, float or string.
