Text Face

Sets the face of the text to be drawn

Syntax

Text Face face

Description

face is the face of the text to be drawn.

The face is built form these components:

1 - Bold
2 - Italic
4 - Underline
8 - Outline
16 - Shadow
32 - Condense
64 - E x t e n d

In order to draw text in both bold and shadow this command would have to be written:

Text Face 1+16
