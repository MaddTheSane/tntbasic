Sprite Y Scale

Gets the y scale of a sprite.

Syntax

scale = Sprite Y Scale (spriteID)

Description

The y scale of a sprite controls the height of the sprite.

spriteID is the number of the sprite.

scale is the scale adjustment the sprite. This value is a floating point mutliplier so if you pass 2 it will double the height of the sprite, if you pass 0.5 it will half the height of the sprite.
