Draw Picture

Draws a picture

Syntax

Draw Picture picID
Draw Picture picID,canvas
Draw Picture picID,x,y

Description

picId can be either an integer or a string. It is the id or the name of the 'PICT' resource to be drawn.

canvas is the id of the canvas the picture should be drawn into.

x and y are the co-ordinates in the canvas of the top left corner of the picture.
