Set Networking Normal

This command sets the networking system to send 'normal' messages.

Syntax

Set Networking Normal

Description

'Normal' message mode is generally used for games that are to be played over a LAN. There is no confirmation that the message will get there but it is much more efficient and faster than 'Registered' mode.

To switch to 'Registered' mode, use the command Set Networking Registered.
