Image Bank

Gets the bank that a sprite gets its images from.

Syntax

bankNumber = Image Bank (spriteID)

Description

Sprites take images out of an image bank and then draw them to screen. Since multiple image banks can be loaded at any time each sprite must be know which one to get it's images from.

spriteID is the sprite to test.

bankNumber is an integer containing the id of the image bank resource the sprite is using.
