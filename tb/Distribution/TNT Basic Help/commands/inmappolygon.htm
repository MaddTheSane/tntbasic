In Map Polygon

Tests whether the specified TNT Map co-ords are contained inside the specified polygon.

Syntax

boolean = In Map Polygon (x,y,z,inIndex)

Description

inIndex is the numberof the polygon to test. This can be any value between 0 and Map Polygon Count -1.

x,y and z are the co-ords that should be tested to see if the fall within the boundaries of the polygon.

boolean is the result of the operation.

True = The co-ords are in the polygon.
False = The co-ords are not in the polygon.
