Calculate Angle

Calculates the angle from a specified point to another specified point.

Syntax

value = Calculate Angle(startX,startY to endX,endY)

Description

This command takes two points and calculates the angle from the start to the end.

Angles are calculated in a clockwise direction. e.g.

0 is straight up
45 is up-right
90 is right
135 is down-right
180 is straight down
225 is down-left
270 is left
315 is up-left
