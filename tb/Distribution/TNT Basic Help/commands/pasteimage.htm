Paste Image

Draws an image directly into the canvas.

Syntax

Paste Image x,y,image
Paste Image x,y,image,bank

Description

The Sprite command does not affect the canvas directly because it does not draw into it. It only draws to the screen when Draw Frame is called.

This command draws the image into the canvas. Once it has been drawn there it will stay until it is drawn over by something else.

image is the index into the image bank of the image to be drawn.

bank is the bank to get the image out of. The bank must be loaded otherwise nothing will be drawn. If this is not specified then the default bank is used.

x and y are co-ordinates in the canvas of where to place it.
