Draw Tracking

Sets the mode for drawing operations

Syntax

Draw Tracking On
Draw Tracking Off

Description

Draw Tracking On
In this mode, the drawing operations are not shown in the screen until a call to Draw Frame is made. All the drawing operations are displayed immediately. This is the default mode.

Draw Tracking Off
This mode is the same as above except that the areas that are updated with a call to Draw Frame need to be explicitly stated using the Inval Rect command. Only these areas will be copied to the screen.
