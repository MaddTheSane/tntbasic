Set Random Seed

Sets the seed of the random number generator.

Syntax

Set Random Seed seedValue

Description

This command sets the seed of the random number generator. All random numbers are generated depending on this seed.

If the seed is set to the same value then the sequence of numbers produced by the Random function will be the same.

In order to generate truly random numbers the seed should be an unpredictable value. (e.g. the current time in Timer)
