Set Current Map

Make all subsequent map commands refer to this map.

Syntax

Set Current Map resID

Description

resID can be either an integer or a string. It is the id or the name of the map to be used as the current map.

All map commands operate on the currently active map. This command can be used to set which map they will have access to.

The specified map must be currently loaded into memory, otherwise this command will cause an error.
