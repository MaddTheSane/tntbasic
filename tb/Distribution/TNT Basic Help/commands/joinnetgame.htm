Join Net Game

Displays a dialog allowing the player to be gathered into a game.

Syntax

result = Join Net Game()
result = Join Net Game(ipAddress)

Description

This command allows the player to join a network game. If no IP address is specified then the user is allowed to enter one themselves.

The value returned is a boolean stating whether or not the game was started.

False = Game Cancelled or User Kicked Out
True = Game Started
