How To Use Sprites

The common way to implement sprites inside a game is with the following structure...

' Initialisation
Load Images
...

' Main loop
while gameRunning

' Position Sprites
Sprite 0,0,0,0
...

' Update the graphics
Draw Frame

wend

Since the sprites are not rendered directly into the canvas they can be moved around easily just using the Sprite command.

The frame can then be drawn to the screen with all the sprites in the new positions using Draw Frame.
