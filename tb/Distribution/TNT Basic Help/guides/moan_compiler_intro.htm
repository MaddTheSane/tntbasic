Moan Compiler

The Moan Compiler allows you to write miniture programs to control several sprites or viewports simultaneously through the use of Moan Channels.

A moan channel is a mini virtual machine that can have sprites and viewports attached to it, once attached they can be controlled by the channel.

Programs for the moan channel are written in a very simple language, and installed into the moan channels from your main TNT Basic program. They then run in parallel to your main program.

Please see the Moan Tutorial for more information.
